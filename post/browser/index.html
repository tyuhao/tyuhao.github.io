1.3 翻译

chrome 和 edge 都自带了翻译。基于chromium的浏览器例如brave 、centbrowser、360极速浏览器等一般都有网页翻译功能。火狐现在正式版还未有网页翻译功能，不过正在火狐开发团队正在研究利用机器学习实现本地翻译以保护用户隐私。firefox 开发版、nightly、测试版可通过实验功能 about:config 打开,暂不支持中文不多介绍。也可调用 bing 或 yandex 的翻译 api进行翻译。火狐要想达到类似网页翻译功能可安装 翻译网页插件。

5. 火狐样式修改

通过修改浏览器样式文件可以使firefox达到和chrome一样的界面
5.1 方法

  1. 在 Firefox 的地址栏访问 about:config，搜索 toolkit.legacyUserProfileCustomizations.stylesheets，设置为 true
  2. 在 Firefox 的地址栏访问 about:support，选择下方的 Profile Folder，点击 Open Folder。
  3. 创建一个chrome 的文件夹
  4. 在chrome文件夹创建userChrome.css,usercontent.css 文件。
